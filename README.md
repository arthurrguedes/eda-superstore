# Análise Exploratória de Vendas — Superstore

## Objetivo
Realizar uma análise exploratória dos dados de vendas da Superstore, com foco em estatística descritiva, identificação de padrões e comportamento das vendas.

## Dataset
- Fonte: Superstore Dataset
- Registros: ~9.800
- Variáveis principais: Sales, Profit, Category

## Etapas da análise
- Identificação de tipos de variáveis
- Estatísticas descritivas
- Análise de distribuição (histograma e boxplot)
- Comparação de vendas por categoria

## Principais insights
- A variável Sales apresenta forte assimetria positiva, com grande concentração de valores baixos.
- A mediana é mais representativa que a média devido à presença de outliers.
- A categoria Technology concentra as maiores vendas e maior variabilidade.

## Tecnologias utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn

## Como executar
Abra o notebook `eda-superstore-estatistica.ipynb` em um ambiente Jupyter ou Google Colab e baixe e utilize o dataset 'superstore'.
